# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст *курсивом*, необходимо обрамить его звездочками, без пробелов - *  текст *

Выделить текст **полужирным** - обрамить текст двойными звездочками, без пробелов- ** текст **
